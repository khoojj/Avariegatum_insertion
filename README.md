This repository contains the files and scripts used for producing the figures in an up-coming publication.

**/Avariegatum_seq/qPCR/251030_scatterplot_R/  **
Scatter plot of sca12 and gltA copy numbers in field-collected ticks
- [Complete analysis in R markdown](https://khoojj.github.io/Avariegatum_insertion/251017_pca.html)


**/Avariegatum_seq/03_cameroonticks/14_Ra_snp/  ** 
Principal component analysis of SNPs from tick and cell line reads mapped to R.africae ESF-5 chromosome

**/Avariegatum_seq/03_cameroonticks/15_Ra_coverage/ **   
Coverage plot for reads from ticks and cell lines mapped to R.africae ESF-5 chromosome
